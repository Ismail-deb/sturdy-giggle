# Documentation Folder

This folder contains additional documentation and resources for the EcoView Greenhouse Monitoring System.

## Contents

### Main Documentation (in root directory)
- **README.md** - Project overview and quick start guide
- **DEPLOYMENT_GUIDE.md** - Comprehensive deployment instructions
- **USER_MANUAL_COMPLETE.md** - Complete user manual with screenshots

### Secondary Documentation (in this folder)
- **COLOR_CODING_README.md** - Sensor status color coding reference
- **DOCUMENTATION_README.md** - Documentation structure guide
- **GEMINI_STATUS.md** - AI integration status and configuration
- **MQTT_BRIDGE_README.md** - MQTT bridge setup and usage
- **RENDER_DEPLOYMENT_GUIDE.md** - Cloud deployment guide for Render.com
- **SCREENSHOT_GUIDE.md** - Screenshot reference guide
- **USER_MANUAL.md** - Standard user manual
- **USER_MANUAL_QUICK.md** - Quick start user guide

### Screenshots
The `screenshots/` subdirectory contains all UI screenshots used in documentation:
- Dashboard views
- Sensor analysis screens
- Settings configuration
- Threshold configuration
- PDF report samples
- Sensor information guides

## Usage

For most users, start with the main documentation in the root directory. The files in this folder provide additional context and detailed information for specific features or deployment scenarios.
